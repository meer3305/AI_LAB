# AI LAB PROGRAMS

This repository contains a comprehensive collection of Artificial Intelligence laboratory programs covering fundamental AI concepts, algorithms, and implementations.

## Contents

### 1. Search Algorithms
- **Breadth-First Search (BFS)** - `search_algorithms/bfs.py`
- **Depth-First Search (DFS)** - `search_algorithms/dfs.py`
- **A* Search Algorithm** - `search_algorithms/a_star.py`
- **Uniform Cost Search** - `search_algorithms/ucs.py`

### 2. Machine Learning Fundamentals
- **Linear Regression** - `machine_learning/linear_regression.py`
- **Logistic Regression** - `machine_learning/logistic_regression.py`
- **K-Means Clustering** - `machine_learning/kmeans.py`
- **Decision Trees** - `machine_learning/decision_tree.py`

### 3. Neural Networks
- **Perceptron** - `neural_networks/perceptron.py`
- **Multi-layer Perceptron** - `neural_networks/mlp.py`
- **Backpropagation** - `neural_networks/backpropagation.py`

### 4. Expert Systems
- **Forward Chaining** - `expert_systems/forward_chaining.py`
- **Backward Chaining** - `expert_systems/backward_chaining.py`
- **Knowledge Base** - `expert_systems/knowledge_base.py`

### 5. Game Theory and Adversarial Search
- **Minimax Algorithm** - `game_theory/minimax.py`
- **Alpha-Beta Pruning** - `game_theory/alpha_beta.py`
- **Tic-Tac-Toe AI** - `game_theory/tic_tac_toe.py`

### 6. Natural Language Processing
- **Text Preprocessing** - `nlp/text_preprocessing.py`
- **N-gram Models** - `nlp/ngram.py`
- **Sentiment Analysis** - `nlp/sentiment_analysis.py`

### 7. Constraint Satisfaction Problems
- **N-Queens Problem** - `csp/n_queens.py`
- **Graph Coloring** - `csp/graph_coloring.py`
- **Sudoku Solver** - `csp/sudoku_solver.py`

## Requirements

```
numpy>=1.21.0
matplotlib>=3.5.0
scikit-learn>=1.0.0
pandas>=1.3.0
```

## Installation

1. Clone the repository:
```bash
git clone https://github.com/meer3305/AI_LAB.git
cd AI_LAB
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Each program can be run independently. For example:

```bash
python search_algorithms/bfs.py
python machine_learning/linear_regression.py
python neural_networks/perceptron.py
```
